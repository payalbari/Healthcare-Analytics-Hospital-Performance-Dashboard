# Healthcare-Analytics-Hospital-Performance-Dashboard
This project presents a Healthcare Analytics Dashboard designed to analyze hospital performance, patient trends, and operational efficiency. 
The dashboard provides meaningful insights using data visualization techniques to support better decision-making in healthcare management.

🚀 Features

📌 Total Patients, Doctors & Appointments Overview

📌 Average Age of Patients

📌 Canceled Appointments Analysis

📌 Revenue & Prescription Insights

📌 Department-wise Performance

📌 Doctor-wise Revenue Tracking

📌 Gender-based Patient Distribution

📌 Monthly Appointment Trends

📌 Top Paying Patients Identification

📌 Treatment Type Analysis

📌 Interactive Filters (Doctor & Department)

📈 Dashboard Insights

The dashboard helps in:

Identifying high-performing departments

Tracking doctor performance and revenue contribution

Understanding patient demographics

Monitoring appointment trends and cancellations

Analyzing treatment frequency and dosage patterns

🛠️ Tools & Technologies Used

Power BI / SQL / Excel

SQL – Data extraction and transformation

Excel / CSV Dataset – Data source

Data Cleaning & Preprocessing

📂 Dataset Description


The dataset includes:


Patient details (age, gender)

Doctor information (department, specialization)

Appointment records

Treatment types

Revenue & billing data

Prescription details
